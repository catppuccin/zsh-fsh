## Usage

1. Download your preferred flavor from the `themes` directory in this repository.
2. Move the files to `~/.config/fsh/`
3. Activate the theme with `fast-theme XDG:catppuccin-mocha` (or `-macchiato`, `-frappe`, `-latte`).
